# ECommerce-Website-ASPNetCore
Full Stack E-Commerce Website with ASP.NET Core, React, Payment Integration and Admin Panel

Project Title → E-Commerce Website (Full Stack)

Tech Stack → ASP.NET Core Web API, ReactJS, SQL Server, Stripe, SendGrid

Features → Admin Panel, Cart, Payment Integration, Email Notifications, Role-Based Login, Orders

Folder Structure

ECommerceAPI/         - Backend (ASP.NET Core Web API)
E-Commerce-Frontend/  - Frontend (ReactJS)
